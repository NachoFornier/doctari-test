# doctari-test

> doctari componenets for testing

[![NPM](https://img.shields.io/npm/v/doctari-test.svg)](https://www.npmjs.com/package/doctari-test) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save doctari-test
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'doctari-test'
import 'doctari-test/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
